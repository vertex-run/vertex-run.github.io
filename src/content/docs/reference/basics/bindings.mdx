---
title: Bindings
description: Immutable and mutable variable bindings in Vertex.
sidebar:
  order: 4
---

```vertex
let name = "Alice"              // immutable — cannot be reassigned
let count: Int = 42             // optional explicit type annotation
mutable let total = 0           // mutable — may be reassigned
total = total + 1               // reassignment (mutable only)
```

Attempting to reassign an immutable binding is a compile-time error (`E101`).

Type annotations are optional when the type can be inferred.
