`debug` is the only way to produce observable output. It is an **expression** — it
evaluates its argument, prints it, and returns the value. It can appear anywhere an
expression is valid.

```vertex
debug "hello world"
debug 42
debug factorial(10)
```

<Aside type="caution">
The checker emits a `W001` warning whenever `debug` appears in source — it is a
development tool, not production output.
</Aside>
