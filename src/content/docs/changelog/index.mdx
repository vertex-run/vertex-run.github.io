Every Vertex release and what changed in it. Newest releases appear first in
the sidebar.

Each entry mirrors the corresponding GitHub Release body, which is the source
of truth. Binaries and installation instructions live on the
[releases page](https://github.com/vertex-run/vertex/releases).

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/):
each release groups its changes under **Added**, **Changed**, **Fixed**, and
**Removed** headings where applicable.
