Released on **2026-04-20**.

### Fixed
- Generated `VtxClosure` classes for closures that capture no variables no
  longer include a redundant empty constructor. Apex auto-supplies a default
  no-arg constructor when none is declared, so emitting one was noise in the
  generated output.
