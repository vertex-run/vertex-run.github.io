Released on **2026-04-18**.

### Added
- Initial public release of the Vertex compiler: lexer, parser, type checker,
  Apex codegen, and tree-walking JIT interpreter.
- CLI entry point `vertex` with `run`, `build`, and `test` subcommands.
- Salesforce integration via `--sf` / `--org` flags.
- Standard library covering numeric, string, list, map, set, option, result,
  date, datetime, and dateformat types.
- Language Server Protocol support with completion, hover, go-to-definition,
  and diagnostics.
- Documentation site at [vertex-run.github.io](https://vertex-run.github.io).
